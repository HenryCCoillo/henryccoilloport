About Me

Soy un desarrollador backend de aplicaciones web escalables y eficientes utilizando tecnologías de Python y PHP. Mi experiencia incluye la creación de sistemas y aplicaciones desde cero, el mantenimiento de aplicaciones existentes y la integración de diferentes sistemas y herramientas para ofrecer soluciones personalizadas a los clientes.
Soy un apasionado de la programación y siempre estoy en busca de nuevas tecnologías y herramientas para mejorar mis habilidades.

Project

Mis proyectos consiste en crear soluciones digitales efectivas y eficientes para las necesidades de los usuarios. Utilizando lenguajes de programación, herramientas y tecnologías de última generación, busco diseñar y desarrollar aplicaciones web intuitivas y atractivas que brinden una excelente experiencia al usuario.

POKE API Frontend

El proyecto en frontend basado en la Poke API, incluye un diseño atractivo para la página de Sign in y Sign up de usuarios. Además de permitir la visualización detallada de información sobre diferentes pokémon,como sus habilidades, estadísticas, y evoluciones.

POKE API Backend

El proyecto Poke API utilizando Django como fuente de datos en el backend. Esta implementado la autenticación de JWT para la seguridad del proyecto y agregué endpoints para listar tanto los Pokémon como sus habilidades correspondientes. Además, documenté el proyecto utilizando Swagger para una mejor comprensión y facilidad de uso.

Sistema de Pago y Servicio API Frontend

En el proyecto que realicé, diseñé e implementé la interfaz de usuario en el frontend de un sistema de pago y servicios, incluyendo una página de inicio de sesión y una lista de pagos y servicios solicitados para cada usuario

Sistema de Pago y Servicio API Backend

El proyecto sistema de pago y servicios en backend utilizando el framework Django, y añadí autenticación JWT y un sistema de pago con expiración. Además, agregué Swagger para documentar la API del proyecto
